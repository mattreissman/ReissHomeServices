# Reiss Home Services Website Refresh

This updated version turns the original single-page site into a cleaner, more polished static website with:

- stronger visual hierarchy
- smoother section-to-section flow
- subtle entrance and hover animations
- mobile navigation
- improved use of the provided project photos
- a more professional quote/contact section
- fixed local asset paths for easier hosting

## File structure

- `index.html` — main page content
- `assets/styles.css` — all styling
- `assets/script.js` — navigation, scroll reveal, quote form behavior
- `assets/` — optimized logo and project images used by the refreshed site
